XPN BUILDER V1 — Final Release
================================

Files in this folder:
  mpc_expansion_builder.html  — Main app (open in Chrome)
  xpn_manual_v1.html          — Trilingual manual ES / EN / FR

How to use:
  1. Open mpc_expansion_builder.html in Chrome
  2. The app runs 100% offline — no internet needed
  3. Click ⊞ Tests (bottom right) to run the 62-test suite
  4. Click ⊞ Pattern to open the Pattern Editor
  5. Click CSTM mode to use the Custom layout editor

Features:
  • 8 banks × 16 pads = 128 total pads
  • 5 modes: DRUM · BEAT · LIVE · H·HOP · ELEC
  • Custom mode (CSTM) with drag & drop
  • New slot type creator (+)
  • Smart auto-assign with synonym detection
  • Pattern Editor with live sequencer
  • Demo preview track (upload or auto-generate)
  • Effects chain per pad (6 types)
  • Export .zip — MPC Live / X / One compatible
  • 62/62 tests passing
