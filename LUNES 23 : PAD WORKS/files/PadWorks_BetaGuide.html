01 Installation
1
Download the .dmg file
You received a download link via email. Download PadWorksXPN.dmg to your Downloads folder.
2
Open the .dmg
Double-click the .dmg file. A window opens showing the Pad Works XPN icon and your Applications folder.
3
Drag to Applications
Drag the Pad Works XPN icon into the Applications folder. Wait for the copy to complete.
4
First launch — bypass Gatekeeper
macOS may block the app on first open. Go to System Settings → Privacy & Security → scroll down → click Open Anyway. This only happens once.
macOS requirement: Ventura 13+ or Sonoma 14+. Intel and Apple Silicon (M1/M2/M3) both supported.
02 First Launch
1
Choose Your MPC
On first launch, a setup screen appears. Select your MPC model: MPC Live 1/2 · MPC One · MPC X · MPC Live 3 · Akai Force
2
Choose Starting Mode
Select your default kit mode. Each mode shows a visual 4×4 pad layout with color-coded categories. BEAT is recommended for first use. Select CUSTOM to build from scratch.
3
Press START
Both device and mode must be selected for START to activate. Your selection is saved — you won't see this screen again unless you click SETUP in the top bar.
Tip: You can change your MPC target device anytime from the top bar dropdown without going back to the setup screen.

04 Key Features
⚡
Smart Auto-Assign
Reads filenames and automatically maps kick, snare, hats, etc. to the correct pads. 3 modes: LONG, KIT, FILL.
🎨
Template Editor
Customize your pad layout visually. Drag tags onto pads, create custom tags, change category colors with MPC classic presets.
🎛
Sampler Edit
Full per-pad control: LEVEL · PAN · PITCH · Filter · ADSR envelope · Layers (up to 4) · Voice mode.
🥁
Pattern Sequencer
8 sequences per bank, swing control, per-step velocity, bank switching with quantization.
🏦
8 Banks × 16 Pads
128 total pad slots. No-duplicate mode ensures each bank has unique samples by default.
📦
Dual Firmware Export
Separate export buttons for MPC 2.x and 3.x. Generates .xpm, .wav, .mpcpattern, .mid and preview audio.
05 Exporting to MPC
1
Name your expansion
Enter a name in the EXPANSION NAME field in the top bar. This becomes the kit filename on the MPC.
2
Check firmware version
On your MPC: Menu → About — check if you're on firmware 2.x or 3.x. Use the matching export button.
3
Click EXPORT FOR MPC 2.X or 3.X
A .zip file is generated and downloaded. All banks, samples, patterns and preview are included.
4
Copy to MPC storage
Unzip the file. Copy the entire folder to your MPC's internal storage or SD card under Expansions/. On the MPC: Menu → Expansions → Browse.
Important: Do not rename folders or files after exporting. The MPC reads the exact structure Pad Works generates.
